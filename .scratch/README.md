# Local work items

This directory is the repository's local Markdown issue tracker.

Create one directory per effort:

```text
.scratch/<feature-slug>/
  spec.md
  issues/
    01-first-vertical-slice.md
    02-next-vertical-slice.md
```

See `docs/agents/issue-tracker.md` for ownership and lifecycle rules. Do not put planning-with-files progress logs here.

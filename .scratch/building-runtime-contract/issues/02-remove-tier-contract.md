# 02 — 从运行时地点档案中删除访古等级

**要实现的内容：** 数据维护者获得不再包含主观价值等级的运行时地点档案；现有全部记录、领域声明和基础校验边界同步完成迁移，而页面行为保持上一阶段已经验收的中立状态。

**前置任务：** 01 — 移除访古等级对浏览体验的影响

**状态：** ready-for-agent

- [ ] 全部运行时地点档案删除 `tier`，除此之外不重排记录、不改写内容，也不顺便清理其他数据字段。
- [ ] `Building` 领域声明、旧等级常量、统计派生和基础 validator 不再要求或表达访古等级。
- [ ] 仓库中的运行时代码不再读取 `tier`，但历史文档中为解释迁移而保留的文字不被误当作代码遗留。
- [ ] validator 的有效样本不再需要 `tier`，真实运行时 artifact 继续通过命令行数据校验。
- [ ] 数据 diff 可被审核为单一、机械的字段删除；记录数量、ID、顺序和其他字段值保持不变。
- [ ] 本地聚合检查与生产构建通过。
